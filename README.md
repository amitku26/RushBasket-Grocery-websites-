# MERN GroceryWebsites | RushBasket

## Overview

A full-stack grocery e-commerce platform built with the MERN stack that allows users to browse products, manage cart, and place orders.

This project is deployed as two separate applications:

## Features
 - ### Admin Pannel
Add new grocery products
Upload product images to Cloudinary
Edit product details
Delete products
##Inventory Control
Manage stock & pricing
Category management

-### User / Customer Features

User Authentication
Secure signup & login
Product Browsing
View grocery items category-wise
Search products by name
Filter products based on price / category
Cart Management
Add items to cart
Increase / decrease quantity
Remove single item or clear full cart
Cart data persists during session
Order Workflow
Checkout page for placing orders
View order summary before confirmation
Responsive UI
Mobile, tablet, and desktop friendly design
Clean gradient modern interface using Tailwind

## Technologies Used
   - ### Frontend: React, TailwindCSS
   - ### Backend: Expressjs, node, mongodb
   - ### Deployment: Render
